Diaphoneme Swap

Diaphoneme Swap was created based off of the work of phonetician John C. Wells. Dr. Wells has given me permission to make this work public.

Demo

The original project is written in SWI-Prolog. In order to demo this in browser three things had to be done.

  1. Refactor the Prolog code
    • SWI-Prolog contained operators not compatible with the ISO Compliant transpiler used for this demo.
  2. Transpile the Prolog code into Javascript
    • Prolog code cannot run natively in a browser.
  3. Create an interface for the transpiled Prolog "backend".
    • Prolog is often run in a console, so to make this demo more user friendly, a UI is being developed.
